# Plug Penguin Game

This is a fresh start for the Plug Penguin game, built with React, Three.js, and React Three Fiber.

## Project Structure

- `components/plug-penguin/` - Main game components
  - `scenes/` - Game scenes
  - `entities/` - Game entities (characters, objects)
  - `hooks/` - Custom hooks for game functionality
  - `utils/` - Utility functions
  - `core/` - Core game systems
  - `ui/` - UI components

## Assets

- `public/models/POOKIE.glb` - The Pookie character model

## Development

This is a simplified version of the game with basic functionality. The game is rendered using React Three Fiber and uses the Pookie model for the main character.

## Features

- Basic 3D scene with lighting and environment
- Pookie character model with simple animation
- Camera controls for exploring the scene

## Future Development
