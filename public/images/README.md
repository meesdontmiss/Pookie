# Club Pookie Images

This directory contains image assets for the Club Pookie virtual world. Replace these placeholder files with your own custom Pookie-themed images.

## Required Images

### Core Assets
- `logo.png` - The Club Pookie logo (recommended size: 200x200px)
- `favicon.ico` - Browser tab icon (32x32px)
- `og-image.jpg` - Open Graph image for social media sharing (1200x630px)

### UI Elements
- `button-background.png` - Background texture for buttons
- `panel-background.png` - Background texture for UI panels
- `loading-spinner.gif` - Loading animation

### Badges
- `nft-badge.png` - Badge for NFT holders (64x64px)
- `token-badge.png` - Badge for token holders (64x64px)
- `admin-badge.png` - Badge for admins (64x64px)

## Image Guidelines

- Use PNG format for images with transparency
- Keep file sizes small for better performance (compress when possible)
- Use consistent style across all images
- For best results, use the recommended dimensions

## Adding Custom Images

1. Create your custom images using design software like Photoshop, GIMP, or Figma
2. Save them with the appropriate filenames listed above
3. Place them in this directory, replacing any placeholder files
4. The application will automatically use your custom images

## Placeholder Images

Until you add your own images, the application will use default placeholder graphics.
